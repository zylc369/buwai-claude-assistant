# 用户交互文档

> **用途**: 描述用户操作流程和交互规范

---

## 1. 用户流程

### 1.1 项目选择流程

```
用户打开页面
    │
    ▼
TopBar显示 "Select Project"
    │
    ├─→ 点击下拉 → 选择已有项目 → 自动选中
    │
    └─→ 点击 "New Project" → 弹出对话框
                              │
                              ├─→ 输入项目名(必填)
                              ├─→ 输入Worktree路径(可选)
                              └─→ 点击 Create → 创建并自动选中
```

### 1.2 工作空间创建流程

```
用户选中项目
    │
    ▼
Sidebar Workspaces区域显示 "+" 按钮
    │
    ▼
点击 "+" → 弹出对话框
    │
    ├─→ 输入工作空间名(必填)
    ├─→ 输入分支名(可选)
    └─→ 点击 Create → 创建并自动选中
```

### 1.3 会话创建流程

```
用户选中工作空间
    │
    ▼
Sidebar Sessions区域显示 "+" 按钮
    │
    ▼
点击 "+" → 无弹窗，直接创建
    │
    ├─→ 自动生成 session_unique_id (UUID)
    ├─→ 自动生成 external_session_id (UUIDv7)
    ├─→ 设置 title = "New Chat"
    └─→ 自动选中新会话
```

**重要**: 会话创建无弹窗，点击即创建。

### 1.4 聊天交互流程

```
用户选中会话
    │
    ▼
ChatInterface 显示聊天区域
    │
    ├─→ 加载历史消息
    └─→ 显示输入框
    │
    ▼
用户输入消息
    │
    ├─→ 按 Enter 发送
    └─→ 按 ⌘/Shift+Enter 换行
    │
    ▼
发送到后端
    │
    ├─→ 流式模式: 实时显示响应
    └─→ 轮询模式: 每2秒检查新消息
```

## 2. 通信模式

### 2.1 轮询模式 (默认)

| 配置 | 值 |
|------|-----|
| 间隔 | 2000ms (2秒) |
| 存储 | localStorage `buwai-chat-mode` |
| 图标 | Radio |

**行为**:
1. 初始加载会话所有消息
2. 记录 lastMessageId
3. 每2秒请求 `id > lastMessageId` 的消息
4. 新消息追加到列表

### 2.2 流式模式

| 配置 | 值 |
|------|-----|
| 协议 | Server-Sent Events (SSE) |
| 存储 | localStorage `buwai-chat-mode` |
| 图标 | Wifi |

**行为**:
1. 建立SSE连接
2. 实时接收chunk事件
3. 累积显示流式内容
4. 收到done事件后显示完整响应

### 2.3 模式切换

- ChatInterface顶部显示模式切换按钮
- 切换后自动保存到localStorage
- 页面刷新后保持之前的选择

## 3. 快捷键

### 3.1 消息输入

| 快捷键 | 行为 |
|--------|------|
| Enter | 发送消息 |
| ⌘ + Enter | 换行 |
| Shift + Enter | 换行 |

### 3.2 禁用条件

以下情况禁用发送:
- 未选中会话
- 正在发送中
- 输入框为空

## 4. 状态持久化

### 4.1 选中的ID

**存储**: localStorage `buwai-persisted-ids`

```json
{
  "project_unique_id": "xxx",
  "workspace_unique_id": "xxx",
  "session_unique_id": "xxx"
}
```

**恢复逻辑**:
1. 页面加载时读取localStorage
2. 恢复选中的项目 → 加载工作空间
3. 恢复选中的工作空间 → 加载会话
4. 恢复选中的会话 → 加载消息

### 4.2 聊天模式

**存储**: localStorage `buwai-chat-mode`

```json
"polling" | "streaming"
```

## 5. UI反馈

### 5.1 加载状态

| 场景 | 显示 |
|------|------|
| 加载项目列表 | "Loading..." |
| 加载消息 | 旋转加载图标 |
| 创建中 | "Creating..." 按钮 |
| 发送中 | 旋转发送按钮 |

### 5.2 空状态

| 场景 | 显示 |
|------|------|
| 无项目 | 下拉显示 "No projects available" |
| 无工作空间 | "No workspaces" |
| 无会话 | "No sessions yet" |
| 无消息 | "No messages yet" |
| 未选工作空间 | "Select workspace to start" |

### 5.3 错误处理

| 场景 | 处理 |
|------|------|
| API错误 | console.error + 恢复UI |
| SSE错误 | console.error + 触发onStreamEnd |
| 网络错误 | 捕获并记录 |

## 6. 时间格式化

**会话时间显示**:

| 时间差 | 显示 |
|--------|------|
| < 1小时 | "Just now" |
| < 24小时 | "{X}h ago" |
| < 7天 | "{X}d ago" |
| >= 7天 | 日期格式 |

**消息时间显示**: 本地时间格式 (HH:MM)

## 8. 验证清单

使用本文档验证交互时:

- [ ] 项目创建时Worktree字段是否可选
- [ ] 会话创建是否无弹窗
- [ ] 会话创建是否生成UUIDv7作为external_session_id
- [ ] 轮询模式是否2秒间隔
- [ ] 增量消息获取是否使用last_message_id
- [ ] 模式切换是否保存到localStorage
- [ ] 选中状态是否持久化并恢复

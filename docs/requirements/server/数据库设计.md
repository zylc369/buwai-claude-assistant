## 实体关系图

```
┌─────────────┐      1:N      ┌─────────────┐
│   Project   │──────────────▶│  Workspace  │
└─────────────┘               └─────────────┘
       │                              │
       │ 1:N                          │ 1:N
       │                              │
       ▼                              ▼
┌─────────────┐               ┌─────────────┐
│   Session   │◀──────────────│   Session   │
└─────────────┘               └─────────────┘
       │
       │ 1:N
       ▼
┌─────────────┐
│   Message   │
└─────────────┘
```

**级联删除规则**:
- 删除 Project → 级联删除所有 Workspace 和 Session
- 删除 Workspace → 级联删除所有 Session
- 删除 Session → 级联删除所有 Message

## 表结构定义

### 1. project 表

**用途**: 项目表，存储项目基本信息

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | Integer | PRIMARY KEY, AUTOINCREMENT | 内部ID |
| `project_unique_id` | Text | UNIQUE, NOT NULL | 项目唯一标识 |
| `worktree` | Text | NOT NULL | 工作树路径 |
| `branch` | Text | NULLABLE | Git分支名称 |
| `name` | Text | NULLABLE | 项目名称 |
| `time_initialized` | Integer | NULLABLE | 初始化时间戳(Unix) |
| `time_created` | Integer | NOT NULL | 创建时间戳(Unix) |
| `time_updated` | Integer | NOT NULL | 更新时间戳(Unix) |

**关系**:
- `workspaces`: 1:N → Workspace (级联删除)
- `sessions`: 1:N → Session (级联删除)

**验证代码**:
```python
# server/database/models.py:15-31
class Project(Base):
    __tablename__ = "project"
    id = Column(Integer, primary_key=True, autoincrement=True)
    project_unique_id = Column(Text, unique=True, nullable=False)
    worktree = Column(Text, nullable=False)
    branch = Column(Text, nullable=True)
    name = Column(Text, nullable=True)
    time_initialized = Column(Integer, nullable=True)
    time_created = Column(Integer, nullable=False)
    time_updated = Column(Integer, nullable=False)
```

### 2. workspace 表

**用途**: 工作空间表，存储项目的工作空间信息

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | Integer | PRIMARY KEY, AUTOINCREMENT | 内部ID |
| `workspace_unique_id` | Text | UNIQUE, NOT NULL | 工作空间唯一标识 |
| `branch` | Text | NULLABLE | Git分支名称 |
| `name` | Text | NULLABLE | 工作空间名称 |
| `directory` | Text | NULLABLE | 工作目录 |
| `extra` | Text | NULLABLE | 额外信息(JSON) |
| `project_unique_id` | Text | NOT NULL, FK | 所属项目ID |

**外键**:
- `project_unique_id` → `project.project_unique_id` (ON DELETE CASCADE)

**索引**:
- `workspace_project_unique_idx` on `project_unique_id`

**验证代码**:
```python
# server/database/models.py:34-58
class Workspace(Base):
    __tablename__ = "workspace"
    id = Column(Integer, primary_key=True, autoincrement=True)
    workspace_unique_id = Column(Text, unique=True, nullable=False)
    branch = Column(Text, nullable=True)
    name = Column(Text, nullable=True)
    directory = Column(Text, nullable=True)
    extra = Column(Text, nullable=True)
    project_unique_id = Column(
        Text,
        ForeignKey("project.project_unique_id", ondelete="CASCADE"),
        nullable=False,
    )
    __table_args__ = (
        Index("workspace_project_unique_idx", "project_unique_id"),
    )
```

### 3. session 表

**用途**: 会话表，存储对话会话信息

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | Integer | PRIMARY KEY, AUTOINCREMENT | 内部ID |
| `session_unique_id` | Text | UNIQUE, NOT NULL | 会话唯一标识 |
| `external_session_id` | Text | NOT NULL | 外部会话ID(前端生成UUIDv7) |
| `sdk_session_id` | Text | NULLABLE | SDK会话ID(Claude返回) |
| `project_unique_id` | Text | NOT NULL, FK | 所属项目ID |
| `workspace_unique_id` | Text | NOT NULL, FK | 所属工作空间ID |
| `directory` | Text | NOT NULL | 工作目录 |
| `title` | Text | NOT NULL | 会话标题 |
| `time_created` | Integer | NOT NULL | 创建时间戳(Unix) |
| `time_updated` | Integer | NOT NULL | 更新时间戳(Unix) |
| `time_compacting` | Integer | NULLABLE | 压缩时间戳 |
| `time_archived` | Integer | NULLABLE | 归档时间戳(NULL=未归档) |

**外键**:
- `project_unique_id` → `project.project_unique_id` (ON DELETE CASCADE)
- `workspace_unique_id` → `workspace.workspace_unique_id`

**索引**:
- `session_project_unique_idx` on `project_unique_id`
- `session_workspace_unique_idx` on `workspace_unique_id`

**验证代码**:
```python
# server/database/models.py:61-96
class Session(Base):
    __tablename__ = "session"
    id = Column(Integer, primary_key=True, autoincrement=True)
    session_unique_id = Column(Text, unique=True, nullable=False)
    external_session_id = Column(Text, nullable=False)  # 新增字段
    sdk_session_id = Column(Text, nullable=True)        # 新增字段
    project_unique_id = Column(
        Text,
        ForeignKey("project.project_unique_id", ondelete="CASCADE"),
        nullable=False,
    )
    workspace_unique_id = Column(
        Text,
        ForeignKey("workspace.workspace_unique_id"),
        nullable=False,
    )
    directory = Column(Text, nullable=False)
    title = Column(Text, nullable=False)
    time_created = Column(Integer, nullable=False)
    time_updated = Column(Integer, nullable=False)
    time_compacting = Column(Integer, nullable=True)
    time_archived = Column(Integer, nullable=True)
```

### 4. message 表

**用途**: 消息表，存储对话中的消息

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| `id` | Integer | PRIMARY KEY, AUTOINCREMENT | 内部ID |
| `message_unique_id` | Text | UNIQUE, NOT NULL | 消息唯一标识 |
| `session_unique_id` | Text | NOT NULL, FK | 所属会话ID |
| `time_created` | Integer | NOT NULL | 创建时间戳(Unix) |
| `time_updated` | Integer | NOT NULL | 更新时间戳(Unix) |
| `data` | Text | NOT NULL | 消息数据(JSON格式) |

**外键**:
- `session_unique_id` → `session.session_unique_id` (ON DELETE CASCADE)

**索引**:
- `message_session_unique_idx` on `session_unique_id`

**data字段JSON格式**:
```json
{
  "role": "user" | "assistant",
  "content": "消息内容"
}
```

**验证代码**:
```python
# server/database/models.py:99-121
class Message(Base):
    __tablename__ = "message"
    id = Column(Integer, primary_key=True, autoincrement=True)
    message_unique_id = Column(Text, unique=True, nullable=False)
    session_unique_id = Column(
        Text,
        ForeignKey("session.session_unique_id", ondelete="CASCADE"),
        nullable=False,
    )
    time_created = Column(Integer, nullable=False)
    time_updated = Column(Integer, nullable=False)
    data = Column(Text, nullable=False)
```

## 数据库配置

| 配置项 | 值 | 说明 |

| WAL模式 | 启用 | 提高并发性能 |

### 1.4 MessageService

**文件**: `server/services/message_service.py`

| 方法 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `create_message` | message_unique_id, session_unique_id, data, time_created, time_updated | Message | 创建消息 |
| `get_message_by_id` | message_id | Optional[Message] | 按ID查询 |
| `get_message_by_unique_id` | message_unique_id | Optional[Message] | 按唯一ID查询 |
| `list_messages` | session_unique_id, offset, limit | List[Message] | 分页查询 |
| `list_messages_after_id` | session_unique_id, last_message_id, limit | List[Message] | 增量获取 |
| `send_ai_prompt` | prompt, session_unique_id, client_config, pool | AsyncIterator[Any] | 发送AI提示词(流式) |
| `count_messages` | session_unique_id | int | 统计消息数 |
| `update_message` | message_unique_id, data, **kwargs | Optional[Message] | 更新消息 |
| `delete_message` | message_unique_id | bool | 删除消息 |

**send_ai_prompt 行为**:
1. 创建用户消息并存储到数据库
2. 调用Claude SDK发送提示词
3. 流式yield响应块
4. 完成后创建AI消息并存储

## 3. 事务管理

**Service层负责事务管理**:

```python
# 示例: MessageService.send_ai_prompt
async def send_ai_prompt(...):
    # 1. 创建用户消息
    await self.message_repo.create(...)
    await self.session.commit()  # 提交事务
    
    # 2. 流式获取AI响应
    async for chunk in ...:
        yield chunk
    
    # 3. 创建AI消息
    await self.message_repo.create(...)
    await self.session.commit()  # 提交事务
```

## 4. 错误处理

| 场景 | 处理方式 |
|------|----------|
| 记录不存在 | 返回 `None` 或 `[]` |
| 唯一约束冲突 | 抛出数据库异常 |
| 外键约束失败 | 抛出数据库异常 |
| 网络错误 | Service层捕获并处理 |

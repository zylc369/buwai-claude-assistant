## 2. React Query Hooks

### 2.1 useProjects

**文件**: `web/hooks/useProjects.ts`

| Hook | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| useProjects | ownerId?: number | UseQueryResult<Project[]> | 获取项目列表 |
| useProject | id: number | UseQueryResult<Project> | 获取单个项目 |
| useCreateProject | - | UseMutationResult | 创建项目 |
| useUpdateProject | - | UseMutationResult | 更新项目 |
| useDeleteProject | - | UseMutationResult | 删除项目 |

**Query Key**: `['projects', ownerId]`

### 2.2 useWorkspaces

**文件**: `web/hooks/useWorkspaces.ts`

| Hook | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| useWorkspaces | projectUniqueId?: string | UseQueryResult<Workspace[]> | 获取工作空间列表 |
| useWorkspace | workspaceUniqueId: string | UseQueryResult<Workspace> | 获取单个工作空间 |
| useCreateWorkspace | - | UseMutationResult | 创建工作空间 |
| useUpdateWorkspace | - | UseMutationResult | 更新工作空间 |
| useDeleteWorkspace | - | UseMutationResult | 删除工作空间 |

**Query Key**: `['workspaces', projectUniqueId]`

### 2.3 useSessions

**文件**: `web/hooks/useSessions.ts`

| Hook | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| useSessions | ListSessionsParams | UseQueryResult<Session[]> | 获取会话列表 |
| useSession | sessionUniqueId: string | UseQueryResult<Session> | 获取单个会话 |
| useSessionByExternalId | externalSessionId: string | UseQueryResult<Session> | 按外部ID获取 |
| useCreateSession | - | UseMutationResult | 创建会话 |

**Query Key**: `['sessions', projectUniqueId, workspaceUniqueId, externalSessionId]`

### 2.4 useMessages

**文件**: `web/hooks/useMessages.ts`

| Hook | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| useMessages | ListMessagesParams | UseQueryResult<Message[]> | 获取消息列表 |
| useMessage | messageUniqueId: string | UseQueryResult<Message> | 获取单个消息 |
| useCreateMessage | - | UseMutationResult | 创建消息 |

**Query Key**: `['messages', sessionUniqueId, offset, limit, lastMessageId]`

**ListMessagesParams**:
```typescript
interface ListMessagesParams {
  session_unique_id: string;
  offset?: number;
  limit?: number;
  last_message_id?: number;  // 增量获取
}
```

## 5. 类型定义

**文件**: `web/lib/api/types.ts`

### Session
```typescript
interface Session {
  id: number;
  session_unique_id: string;
  external_session_id: string;     // 前端生成UUIDv7
  sdk_session_id?: string | null;  // Claude SDK返回
  project_unique_id: string;
  workspace_unique_id: string;
  directory: string;
  title: string;
  time_created: number;
  time_updated: number;
  time_compacting: number | null;
  time_archived: number | null;
}
```

### CreateSessionRequest
```typescript
interface CreateSessionRequest {
  session_unique_id: string;
  external_session_id: string;     // 必填
  project_unique_id: string;
  workspace_unique_id: string;
  directory: string;
  title: string;
}
```

### Message
```typescript
interface Message {
  id: number;
  message_unique_id: string;
  session_unique_id: string;
  time_created: number;
  time_updated: number;
  data: string;  // JSON string
}
```

### SSEEvent
```typescript
interface SSEEvent {
  type: 'chunk' | 'done' | 'error';
  content?: unknown;
  session_unique_id?: string;
  sdk_session_id?: string;
  message?: string;
}
```

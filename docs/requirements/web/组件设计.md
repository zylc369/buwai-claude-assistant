### 2.2 TopBar

**文件**: `web/components/layout/TopBar.tsx`

**功能**:
- 显示Logo和应用名称
- 项目下拉选择器
- 创建新项目对话框

**状态**:
| 状态 | 类型 | 说明 |
|------|------|------|
| isCreateDialogOpen | boolean | 对话框开关 |
| newProjectName | string | 新项目名 |
| newProjectWorktree | string | 工作树路径(可选) |

**创建项目对话框字段**:
| 字段 | 必填 | 说明 |
|------|------|------|
| Project Name | 是 | 项目名称 |
| Worktree Path | 否 | 工作树路径 |

**验证代码**:
```tsx
// web/components/layout/TopBar.tsx:28-43
const handleCreateProject = async () => {
  if (!newProjectName.trim()) return;  // worktree可选
  const project = await createProject.mutateAsync({
    project_unique_id: crypto.randomUUID(),
    worktree: newProjectWorktree.trim() || null,
    name: newProjectName.trim(),
  });
  setSelectedProject(project);
};
```

### 2.3 Sidebar

**文件**: `web/components/layout/Sidebar.tsx`

- 显示会话列表

**创建工作空间对话框字段**:
| 字段 | 必填 | 说明 |
|------|------|------|
| Workspace Name | 是 | 工作空间名称 |
| Branch | 否 | Git分支 |

**创建会话**:
- 无弹窗

**验证代码**:
```tsx
// web/components/layout/Sidebar.tsx:49-68
const handleCreateSession = async () => {
  const externalSessionId = uuidv7();  // UUIDv7生成
  const newSession = await createSession.mutateAsync({
    session_unique_id: crypto.randomUUID(),
    external_session_id: externalSessionId,
    project_unique_id: selectedProject.project_unique_id,
    workspace_unique_id: selectedWorkspace.workspace_unique_id,
    directory: directory,
    title: "New Chat",
  });
  setSelectedSession(newSession);
};
```

### 2.4 ChatInterface

**文件**: `web/components/layout/ChatInterface.tsx`

**功能**:
- 管理消息状态
- 轮询/流式模式切换
- 加载初始消息
- 增量获取新消息

**Props**: 无(使用Zustand Store)

**状态**:
| 状态 | 类型 | 说明 |
|------|------|------|
| streamingContent | string | 当前流式内容 |
| mode | 'polling' \| 'streaming' | 通信模式 |
| messages | Message[] | 消息列表 |
| lastMessageId | number | 最后消息ID(增量获取) |
| isLoading | boolean | 加载状态 |

**轮询配置**:
- 间隔: 2000ms (2秒)
- localStorage key: `buwai-chat-mode`

**验证代码**:
```tsx
// web/components/layout/ChatInterface.tsx:47-68
useEffect(() => {
  if (mode !== "polling" || !selectedSession?.session_unique_id) return;
  
  pollingIntervalRef.current = setInterval(() => {
    fetchIncrementalMessages(selectedSession.session_unique_id, lastMessageId);
  }, POLLING_INTERVAL_MS);  // 2000ms
  
  return () => clearInterval(pollingIntervalRef.current);
}, [mode, selectedSession?.session_unique_id, lastMessageId]);
```

### 2.5 ChatArea

**文件**: `web/components/layout/ChatArea.tsx`

**Props**:
| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| messages | Message[] | 是 | 消息列表 |
| isLoading | boolean | 否 | 加载状态 |
| streamingContent | string | 否 | 流式内容 |
| sdkSessionId | string | 否 | SDK会话ID |

**功能**:
- 渲染消息列表
- 区分用户/AI消息样式
- 自动滚动到底部
- 显示时间戳

**消息解析**:
```typescript
function parseMessageData(data: string) {
  const parsed = JSON.parse(data);
  return {
    role: parsed.role || "assistant",
    content: typeof parsed.content === "string" 
      ? parsed.content 
      : JSON.stringify(parsed.content),
  };
}
```

### 2.6 InputArea

**文件**: `web/components/InputArea.tsx`

**Props**:
| 属性 | 类型 | 必填 | 说明 |
|------|------|------|------|
| onStreamingContent | (content: string) => void | 否 | 流式内容回调 |
| onStreamEnd | () => void | 否 | 流结束回调 |
| onSdkSessionId | (sdkSessionId: string) => void | 否 | SDK会话ID回调 |

**功能**:
- 多行输入框(自动调整高度，最大200px)
- 发送按钮

**快捷键**:
| 按键 | 行为 |
|------|------|
| Enter | 发送消息 |
| ⌘/Shift + Enter | 换行 |

**验证代码**:
```tsx
// web/components/InputArea.tsx:92-97
const handleKeyDown = (e: KeyboardEvent<HTMLTextAreaElement>) => {
  if (e.key === "Enter" && !e.shiftKey && !(e.metaKey || e.ctrlKey)) {
    e.preventDefault();
    handleSubmit();
  }
};
```

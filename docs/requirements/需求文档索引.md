## 项目概述

Claude Assistant 是一个基于 Claude SDK 的 AI 编程助手应用，采用前后端分离架构：

- **Server**: Python FastAPI 后端，集成 Claude SDK，提供 RESTful API 和 SSE 流式响应
- **Web**: Next.js 前端，提供 ChatGPT 风格的聊天界面

## 技术栈

### Server
- Python 3.11+
- FastAPI + Uvicorn
- SQLAlchemy 2.0 (Async) + aiosqlite
- Alembic (数据库迁移)
- Claude Agent SDK

### Web
- Next.js 16 (App Router)
- React 19 + TypeScript 5
- Tailwind CSS 4
- @base-ui/react (UI组件)
- Zustand (客户端状态)
- @tanstack/react-query (服务端状态)

## 验证清单

4. **交互验证**: 对照 `web/用户交互.md` 执行手动测试
